# LightTraffic 🚦
**A Pure Digital Logic Traffic Light Controller**

Welcome to the **LightTraffic** repository! This project demonstrates the design and simulation of a traffic light control system built entirely using **digital Integrated Circuits (ICs)**. 

Unlike conventional beginner projects, this system operates **without** the use of microcontrollers such as Arduino or ESP32. It relies solely on fundamental digital electronics—utilizing components like 555 timers, counters, flip-flops, and basic logic gates—to manage the precise timing and state sequencing of the red, yellow, and green lights.

### 🛠️ Repository Contents
* `LightTraffic.pdsprj`: The complete circuit simulation file, built and ready to run in **Proteus Design Suite**.
* `ReportIC.pdf`: A comprehensive project report detailing the design methodology, logic gate schematics, timing diagrams, and operational theory.

### 🎯 Key Highlights
* **100% Hardware-Based:** Demonstrates a deep understanding of sequential and combinational logic design.
* **Reliable Timing:** Uses pure electronic components for clock generation and state transitions.
* **Educational Value:** A practical exploration into the core concepts of digital systems before stepping into microcontroller programming.
